User avatar — shows an image, or color-coded initials derived from the name. Optional status dot.

```jsx
<Avatar name="Courtney Leacock" status="online" />
<Avatar name="Jordan Daly" size="lg" status="risk" />
<Avatar src="/me.jpg" name="Me" size="sm" ring />
```

Sizes `xs/sm/md/lg`. Color is deterministic per name (iris/coral/signal/info). Use `status="risk"` for high-risk people in lists.
